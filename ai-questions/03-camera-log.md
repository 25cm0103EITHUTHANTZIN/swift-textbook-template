### Q1

**質問：**

Struct　とはなんですか。

**AIの回答の要点：**

struct（Structure / 構造体）とは、「データ」と「そのデータに関する機能」を1つにまとめるための型 です。

SwiftUIでは画面そのものを struct で作る。

**なぜclassではなくstructをよく使うの？**

Structは　値そのものをコピーする

Classは　同じデータを共有する

- Struct = コピー型（値型）
- Class  = 共有型（参照型）

```swift
struct User {
    var name: String
}
var user1 = User(name: "EI")
var user2 = user1
```
user1 と user2 は別物になる。

変更してみる　→ user2.name = "Taro"

結果：

user1.name → "EI"
user2.name → "Taro"　になる。

なぜ？→ コピーされたから。
